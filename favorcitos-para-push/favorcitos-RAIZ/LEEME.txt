PARA ffaavvoorrcciittooss1 (o cualquier repo nuevo)
====================================================

1. En Vercel, ve a tu proyecto → Settings → Git
   Copia la URL del repo conectado

2. Doble clic en SUBIR_A_REPO.command
   Pega la URL del repo
   Enter

3. En Vercel:
   - Root Directory = VACÍO (borra todo)
   - Redeploy

El proyecto tiene package.json en la raíz.
Root Directory debe estar vacío.
